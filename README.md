SmartStorageManager Project 

Overview:
A storage management system to learn Java fundementals, such as OOP, Scanner class, Arrays, Loops, Read Lines in files, Loading / Saving info into txt files, 


Features:
1: Registering a new customer
2: Display all units
3: Search units by number
4: Filter units vacant/occupied/size
5: Vacate tenant / Delete Unit
7: Generate financial report
8. Save and load data from file

Resources: 
InteliJ IDE
Java
Array List 
File I/O 
Object-Oriented Programming (OOP)
Textbook:Starting Out with Java: Early Objects (6th Edition)
